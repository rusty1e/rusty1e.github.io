部署hexo个人博客并在github托管

git bash下操作

安装hexo

生成博客

github建立仓库

要选Public和Add a README file

生成SSH密钥（RSA）

密钥文件存储在C:\Users\用户名.ssh

id_rsa.pub存储的是公钥

测试是否成功

安装hexo自动部署工具

修改_config.yml

第一次上传要配置email和用户名，用户名要区分大小写

成功的话会弹出来一个窗口进行登录

然后就可以开始部署了
